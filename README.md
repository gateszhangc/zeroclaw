# Zeroclaw

An editorial keyword site about ZeroClaw built with Next.js 16, Tailwind CSS 4, and shadcn/ui.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- shadcn/ui
- Playwright

## Scripts

```bash
npm run brand:generate
npm run dev
npm run lint
npm run build
npm run test:e2e
```

## Brand Assets

`npm run brand:generate` rebuilds the generated logo, favicon, and app icon from the SVG source template in `scripts/generate-brand-assets.mjs`.
